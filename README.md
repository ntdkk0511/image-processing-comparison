# Image Processing Comparison

PythonとOpenCVを用いて、代表的な画像処理アルゴリズムを比較する個人学習用プロジェクト。

## 内容

* グレースケール変換
* Sobelエッジ検出
* Cannyエッジ検出
* 処理時間の比較

## 実装の概要

* 画像処理を関数化し，ウインドウサイズに応じて画像がリサイズされて表示される
* 処理ごとに関数化することで可読性・再利用性が向上する

## 使用技術

* Python
* OpenCV
* NumPy
* Matplotlib



## 実行結果

画像に対して，Sobel法とCanny法の処理にかかった処理時間を以下に示す．

Sobel　0.025701

Canny  0.011667



Cannyエッジ検出の方がSobelエッジ検出より短い処理時間となった．複数回実施し，平均処理時間を比較する必要がある．
